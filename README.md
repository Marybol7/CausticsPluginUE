# CausticsPluginUE
Liquid and Caustics effect in Unreal Engine (to be made plugin). 


TFG – Simulación de efectos de cáusticas y líquidos en tiempo real (Unreal Engine 5)
- Descripción

Proyecto de TFG centrado en la creación de shaders y materiales personalizados en Unreal Engine 5 para simular efectos de cáusticas y líquidos en tiempo real.
El objetivo es recrear el comportamiento óptico del agua (reflexión, refracción y movimiento) con un enfoque optimizado para renderizado en tiempo real.

- Progreso técnico actual

Configuración y aprendizaje avanzado de Unreal Engine 5.

Creación de materiales base y entorno de prueba.

Implementación del shader de cáusticas (M_Caustics_LF):

Uso de nodos TexCoord, Panner, Sine y Lerp para animar el patrón de luz.

Pruebas con World Space UVs para mantener coherencia en geometrías distintas.

Ajuste de intensidad, dirección y velocidad del movimiento.

Integración en una escena de demostración con iluminación dinámica.

Pruebas comparativas entre materiales locales y globales.
